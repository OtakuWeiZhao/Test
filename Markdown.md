# Markdown
Markdown是一种用来写作的轻量级[标记语言]

## Markdown优点
专注你的文字内容而不是排版样式.
轻松导出HTML,PDF和本身的.md文件.
纯文本内容,兼容所有的文本编辑器与字处理软件.
可读,直观.适合所有人的写作语言

## 语法规则
# 一级标题
## 二级标题
### 三级标题
以此类推,总共六级标题,建议在井号后加一个空格

## 列表
熟悉 HTML 的同学肯定知道有序列表与无序列表的区别，在 Markdown 下，列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加 1. 2. 3. 符号要和文字之间加上一个字符的空格。
#### 无序列表
* 1
* 2
* 3

#### 有序列表
1. 1
2. 2
3. 3

##引用
只需在文本前加入>即可
>这里是引用
